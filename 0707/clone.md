# clone 

- 원격저장소를 이용하여 협업을 시작 시 로컬로 저장소를 복제한다.

> clone은 원격저장소를 받아온다. 
>
> pull은 원격저장소의 커밋을 받아온다. 



### 명령어

| 입력 코드                    | 설명                         |
| ---------------------------- | ---------------------------- |
| git clone https://원격저장소 | 원격저장소를 로컬로 받아온다 |

> 가능하면 바탕화면(로컬의 루트 경로)에서 실행하는게 좋다.
